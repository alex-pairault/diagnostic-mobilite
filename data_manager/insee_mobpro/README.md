## MOBPRO

### Description

insee_mobpro folder contains all files to get work flows data from the database, previously stored from a csv file.

### Structure

- _flows_home_work.py_ : main file to manage data
- _save_data_from_csv_to_db.py_ : used once to store data from csv file to database.
- _data/_ : folder with raw csv data
